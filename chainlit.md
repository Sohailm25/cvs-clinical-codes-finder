# Clinical Codes Finder

Search across 6 medical coding systems with intelligent query understanding.

## Supported Systems
- **ICD-10-CM** - Diagnosis codes
- **LOINC** - Lab test codes
- **RxTerms** - Medication codes
- **HCPCS** - Supply/service codes
- **UCUM** - Unit codes
- **HPO** - Phenotype codes

## Example Queries
- `diabetes` - Find diagnosis codes
- `glucose test` - Find lab test codes
- `metformin 500 mg` - Find medication codes
- `wheelchair` - Find supply codes
